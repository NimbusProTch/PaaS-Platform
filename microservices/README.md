# Microservices

This directory contains all microservice implementations for the PaaS Platform.

## Services

- **product-service**: Product management service
- **user-service**: User management service
- **order-service**: Order processing service
- **payment-service**: Payment processing service
- **notification-service**: Notification service
- **ecommerce-platform**: Main ecommerce platform

## Build Status

All services are built automatically via GitHub Actions with multi-arch support (amd64/arm64).
